# Tech Stack - Kaya SmartProcure

- **Frontend**: React 19 (Vite, TypeScript, Tailwind CSS, Lucide Icons, Recharts, Motion)
- **Backend**: Node.js + Express (full-stack API proxy setup)
- **AI Integration**: `@google/genai` TypeScript SDK (utilizing `gemini-3.5-flash` for high-speed audits and sentiment parsing)
- **Offline Storage**: LocalStorage / client-side React Context state cache (enabling robust offline retrieval and fallback operations)
- **Audio Capture**: Standard HTML5 MediaRecorder API (capturing PM voice notes in real-time)
